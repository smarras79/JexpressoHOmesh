## Installing and testing package

Installing package goes same way like other packages in julia, i.e.
```julia
julia> Pkg.add("NodeNumbering")
```

Testing package can be done using `Pkg.test`, i.e.
```julia
julia> Pkg.test("NodeNumbering")
```
